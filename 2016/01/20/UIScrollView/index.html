title: UIScrollView

date: 2016-01-19 22:20:38

tags:

总结：内容的大小和相对位置；滑动和拖拽属性设置；滑动和拖拽的事件响应；滑动条的样式设置；滑动式的动画；

类库API：

  1. 用来展示比屏幕大的内容
  2. 通过滑动和缩放手势进行查看
  3. 三个孩子：tableview, textview, collectionview
  4. 本质是uiview, 是一个whose origin is adjustable over the content view.
  5. 必须知道content size
  6. 展示内容边界：大小和相对于scrollview的偏移量：
    1. contentSize: CGSize
    2. contentOffset: CGPoint
    3. setContentOffset: contentOffset
    4. contentInset: UIEdgeInsets 是个什么东东
  7. 滚动
    1. 能不能滚？ scrollEnabled: Bool
    2. 只能横向或者竖向滚？directionalLockEnabled: Bool
    3. 点击状态栏是否回滚到顶部？（比如微信朋友圈）scrollsToTop: Bool
    4. 是否整页滑动？pagingEnabled: Bool
    5. 是否有缓冲区？bounces: Bool
    6. 滑动的加速度decelerationRate: CGFloat
    7. 用户是否开始滚了dragging: Bool { get }
    8. 用户是否触碰到了content？这时候不一定开始滚了。tracking: Bool { get }
    9. 是否还在滚动？尽管手指可能已离开。decelerating: Bool { get }
    10. scrollRectToVisible(_ rect: CGRect,animated animated: Bool)
    11. touchesShouldBegin(_ touches: Set,withEvent event: UIEvent?,inContentView view: UIView) -> Bool
    12. touchesShouldCancelInContentView(_ view: UIView) -> Bool
    13. canCancelContentTouches: Bool
    14. delaysContentTouches: Bool
    15. ​
  8. 滚动条
    1. 滚动条样式：默认/黑的/白的 indicatorStyle: UIScrollViewIndicatorStyle
    2. 滚动条与边框的距离 scrollIndicatorInsets: UIEdgeInsets
    3. 是否显示横向滚动条？showsHorizontalScrollIndicator: Bool
    4. 是否显示竖向滚动条？showsVerticalScrollIndicator: Bool
    5. flashScrollIndicators()
  9. 放大和缩小
    1. panGestureRecognizer: UIPanGestureRecognizer { get }
    2. pinchGestureRecognizer: UIPinchGestureRecognizer? { get }
    3. zoomToRect(_ rect: CGRect,animated animated: Bool)
    4. 目前伸缩比例 zoomScale: CGFloat
    5. setZoomScale(_ scale: CGFloat,animated animated: Bool)
    6. 缩放最大比 maximumZoomScale: CGFloat
    7. 缩放最小比 minimumZoomScale: CGFloat
    8. zoomBouncing: Bool { get }
    9. zooming: Bool { get }
    10. bouncesZoom: Bool
  10. 状态保留机制：下次打开滚到上次离开时的位置。比如书籍阅读。
  11. scrollview可以设置UIScrollViewKeyboardDismissMode让键盘消失，比如当你的输入框放置在一个scrollview中，这非常有用
  12. UIScrollViewDelegate
    1. 响应滑动和拖拽（这些方法都是Optional）
      1. scrollViewDidScroll
      2. scrollViewWillBeginDragging
      3. scrollViewWillEndDragging
      4. scrollViewShouldScrollToTop
      5. scrollViewDidScrollToTop
      6. scrollViewWillBeginDecelerating
      7. scrollViewDidEndDecelerating
    2. 响应缩放
      1. viewForZoomingInScrollView ->UIView
      2. scrollViewWillBeginZooming
      3. scrollViewDidEndZooming
      4. scrollViewDidZoom
    3. 响应动画
      1. scrollViewDidEndScrollingAnimation
